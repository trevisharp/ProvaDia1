Não tem muito segredo no diagrama de máquina de estados.
É o que a gente falou mesmo. Olha as notas da CIS:
-Adicionar 0,2 para as etapas do fluxo de login.
-Adicionar 0,2 para as transições entre os estados.
-Adicionar 0,2 para especificação das ações associadas corretamente.
-Adicionar 0,2 para as condições ou restrições nas transições.
-Adicionar 0,2 para organização lógica do diagrama.

O mesmo vale para o de atividade:
-Adicionar 0,2 para as etapas do fluxo de login.
-Adicionar 0,2 para as mensagens trocadas entre os objetos.
-Adicionar 0,2 para a organização lógica do diagrama.
-Adicionar 0,2 para a ordem correta das sequências de mensagens.
LEMBRE-SE DE NÃO DEIXAR NENHUMA ATIVIDADE SOLTA, cada atividade deve ligar ou em outra atividade ou no simbolo de fim do diagrama
